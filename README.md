# سرو (SARV) — پروپوزال فنی و مالی سوپر اپلیکیشن هوشمند زیبایی و سلامت

یک وب‌سایت تک‌صفحه‌ای حرفه‌ای (RTL، فارسی) برای ارائه پروپوزال فنی و مالی پروژه **سرو** به کارفرما.
کاملاً استاتیک (HTML/CSS/JS خالص) و آماده انتشار روی **GitHub Pages**.

## ✨ امکانات
- طراحی لوکس و مدرن با تم تیره، گلس‌مورفیسم و انیمیشن‌های نرم
- قیمت‌گذاری کاملاً تفکیک‌شده تا سطح زیرماژول (مجموع کل: **۳٫۵ میلیارد تومان**)
- سوییچ بین نسخه **MVP (۱٫۵ میلیارد)** و **سوپر اپ کامل (۳٫۵ میلیارد)**
- نقشه راه ۵ ماهه به‌صورت چارت گانت + تایم‌لاین
- مقایسه روش‌های هوش مصنوعی (ابری گوگل / سرور اختصاصی)
- پلان پرداخت اقساطی متصل به دستاوردها
- بخش تعاملی «۳ سوال برای شروع» با ساخت خلاصه انتخاب کاربر
- بخش نمونه‌کارها (قابل ویرایش)
- کاملاً ریسپانسیو و سازگار با موبایل

## 🚀 انتشار روی GitHub Pages

### روش ۱ — خودکار (پیشنهادی، با GitHub Actions)
۱. یک ریپازیتوری جدید در گیت‌هاب بسازید (مثلاً `sarv-proposal`).
۲. این دستورها را در پوشه پروژه اجرا کنید:
```bash
git init
git add .
git commit -m "SARV proposal site"
git branch -M main
git remote add origin https://github.com/<USERNAME>/<REPO>.git
git push -u origin main
```
۳. در گیت‌هاب: **Settings → Pages → Build and deployment → Source = GitHub Actions**.
۴. ورک‌فلوی `deploy.yml` به‌صورت خودکار سایت را منتشر می‌کند. آدرس نهایی:
`https://<USERNAME>.github.io/<REPO>/`

### روش ۲ — ساده (انتشار از شاخه)
در **Settings → Pages → Source = Deploy from a branch → Branch = `main` / root** و ذخیره کنید.
